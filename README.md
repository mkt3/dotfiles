# dotfiles

## Supported OS
- macos
- ubuntu

## Install

1. Download

  ```bash
  git clone https://github.com/mkt3/dotfiles.git .dotfiles
  cd .dotfiles
  ```
2. Install
  ```bash
  ./install.sh
  ```
